# Анализ требований: Терминал печати фотографий (PhotoKiosk)

## 1. Формализованные требования

### Глоссарий
* **БТ** – бизнес-требование
* **ПТ** – пользовательское требование
* **ФТ** – функциональное требование
* **БП** – бизнес-правила
* **СТ** – системные требования
* **АК** – атрибуты качества
* **ВСиИ** – внешние системы и интерфейсы
* **Огр** – ограничения

### Требования функционального характера

#### Бизнес-требования
* **БТ001.** Необходимо разработать ПО для автомата, которое позволит пользователям печатать фото из Instagram или с телефона.

#### Пользовательские требования
* **ПТ001.** Пользователь должен иметь возможность печатать фото из Instagram или с телефона.
* **ПТ002.** Пользователь должен иметь возможность авторизоваться в Instagram.
* **ПТ003.** Пользователь должен иметь возможность соединить телефон с автоматом через кабель.
* **ПТ004.** Пользователь должен иметь возможность выбрать количество фотографий для печати через экран автомата.
* **ПТ005.** Пользователь должен иметь возможность оплатить печать по карте через терминал бесконтактной оплаты.

#### Функциональные требования
* **ФТ001.** Система должна предоставить возможность авторизации пользователя в Instagram через телефон или электронную почту.
* **ФТ002.** Система должна предоставить возможность выбрать 2 способа печати: из Instagram или загрузить с телефона.
* **ФТ003.** Система должна предоставить возможность выбрать количество фотографий для печати через экран.
* **ФТ004.** Система должна предоставить возможность оплатить печать по карте через терминал.
* **ФТ005.** Система должна после завершения печати закрывать страницу пользователя в Instagram, если использовался ввод логина.
* **ФТ006.** Система должна после завершения печати сигнализировать о необходимости забрать телефон.

### Требования нефункционального характера

#### Бизнес-правила
* **БП001 - БП007.** Пользователь имеет возможность выбора источника фото, входа в аккаунт, подключения телефона, выбора фото, выбора количества/формата, оплаты картой. Автомат должен обеспечивать безопасность личных данных.

#### Системные требования
* **СТ001. Аппаратное обеспечение:** 8-ядерный процессор, 8 ГБ ОЗУ, 128 ГБ SSD/eMMC, сенсорный экран 7–15 дюймов (Full HD, мультитач), Wi-Fi/Ethernet, термопринтер (5×5 см, 10×15 см), поддержка NFC.
* **СТ002. Программное обеспечение:** ОС с поддержкой драйверов (Linux Ubuntu 20.04 / Raspbian 11 или Windows 10 IoT Core).

#### Атрибуты качества
* **АК001.** Система должна работать без перебоев не менее 99,9% времени.
* **АК002.** Время отклика на касание: не более 200 мс.
* **АК003.** Время подключения к профилю Instagram после ввода логина: не более 10 секунд.
* **АК004.** Загрузка превью фотографий: не более 2 секунд при скорости 10 Мбит/с.

#### Внешние системы и интерфейсы
* **ВСиИ001.** Получение фото из Instagram через Instagram API.
* **ВСиИ002.** Взаимодействие с принтером через протокол ESC/POS.
* **ВСиИ003.** Взаимодействие с терминалом оплаты через протоколы OPOS/JavaPOS.

#### Ограничения
* **Огр001.** Оплата только по карте через терминал бесконтактной оплаты.
* **Огр002.** Лимит печати: от 1 до 100 фото (при наличии бумаги).

## 2. Сценарии использования (Use Cases)

### Сценарий: Выбор фото для печати
1. Пользователь нажимает кнопку «Выбрать фото для печати».
2. **Вариант А (Instagram):** Вводит логин (профиль должен быть открытым) -> Система проверяет доступность фото.
3. **Вариант Б (Телефон):** Подключает телефон через USB -> Выбирает фото из галереи.
4. Пользователь указывает количество фото и копий (1-100).
5. Оплата картой через терминал.
6. Запуск печати после подтверждения платежа.
7. Завершение сеанса (автовыход из Instagram или сигнал отключить телефон).

## 3. Обработка системных ошибок

1. **Ошибки авторизации Instagram:** Закрытый профиль, неверный логин, проблемы с API (уведомления пользователю).
2. **Ошибки подключения телефона:** Неисправный кабель/порт, несовместимость устройства.
3. **Ошибки оплаты:** Неисправность терминала, отказ банка, таймаут оплаты (отмена через 2-3 мин).
4. **Ошибки печати:** Закончилась бумага, замятие бумаги.
5. **Ошибки завершения сеанса:** Телефон не отключен (повторный сигнал), сессия Instagram не закрыта (принудительный выход через 5 мин).
